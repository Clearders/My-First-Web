Web development can feel overwhelming at first — there are dozens of languages, frameworks, and tools
to learn. But the good news is that you only need three things to get started: **HTML**,
**CSS**, and **JavaScript**.

## 1. HTML — The Structure

HTML (HyperText Markup Language) is the skeleton of every web page. It describes the content and
structure using *tags*:

```html
<h1>Hello, World!</h1>
<p>This is my first web page.</p>
<a href="https://example.com">Visit Example</a>
```

## 2. CSS — The Style

CSS (Cascading Style Sheets) makes your page look good. You can control colours, fonts, layout,
and animations:

```css
body {
  font-family: sans-serif;
  background: #f8fafc;
  color: #1e293b;
}

h1 { color: #2563eb; }
```

## 3. JavaScript — The Behaviour

JavaScript brings interactivity. Buttons that respond to clicks, forms that validate input,
pages that update without reloading — all powered by JS.

```js
document.querySelector('button').addEventListener('click', () => {
  alert('Hello from JavaScript!');
});
```

## What's Next?

Once you are comfortable with the basics, explore frameworks like **React** or
**Vue**, and backend technologies like **Node.js**. Most importantly,
*build things* — even small projects teach you more than any tutorial.

> "The best way to learn programming is to write programs." — Dennis Ritchie

Happy coding! 🚀
