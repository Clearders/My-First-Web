CSS has evolved dramatically over the past few years. Here are ten tricks that will save you time
and make your stylesheets cleaner and more powerful.

## 1. CSS Custom Properties

Store values in variables and reuse them everywhere:

```css
:root {
  --primary: #2563eb;
  --radius: 12px;
}

.btn { background: var(--primary); border-radius: var(--radius); }
```

## 2. Logical Properties

Use `margin-inline` and `padding-block` instead of direction-specific shorthands
for better internationalization support.

## 3. Container Queries

Style elements based on their *container's* size, not the viewport:

```css
.card-container { container-type: inline-size; }

@container (min-width: 400px) {
  .card { display: grid; grid-template-columns: 1fr 2fr; }
}
```

## 4. :is() and :where()

Simplify complex selectors:

```css
/* Before */
h1 a, h2 a, h3 a { color: blue; }

/* After */
:is(h1, h2, h3) a { color: blue; }
```

## 5. aspect-ratio

Maintain element proportions with a single line:

```css
.thumbnail { width: 100%; aspect-ratio: 16 / 9; }
```

These are just a taste — CSS is full of gems waiting to be discovered!
